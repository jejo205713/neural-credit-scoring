Credit Risk Analysis Dashboard

Welcome to the CreditRisk AI platform. This system leverages machine learning to analyze user data and predict creditworthiness. Use the tools in the sidebar to train models, evaluate individual user profiles, and manage the dataset.

Project Workflow

1. Data Management: Add new user data to the production dataset via the 'Add User' page.
2. Model Training: Use the 'Train Model' page to train a Random Forest classifier on the latest dataset. This generates analytical graphs on feature importance and model performance.
3. Prediction & Analysis: Evaluate single users with the 'Predict User' tool for a detailed profile analysis, or run batch predictions on the entire dataset with 'Predict All'.
